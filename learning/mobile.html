1. Подготовка

Проверить:

  • Телефон заряжен (powerbank в наличии)
  • Санитарная Книжка
  • Ручка + Блокнот
  • Информационные письма для работы
  • Набор COVID (Маска и Перчатки)

2. Осмотр торгового зала

Первичный

  • Определить количество ОМП/ДМП в ТК.
  • Определить наилучшее место в ОМП для формирования выкладки.

Регулярный

  • Оценить представленность ТМ “Айсберри”, ПХ с учетом достигнутых договоренностей с ЛПР в ТК.

3. Приветствие

Первичный

  1. Познакомиться с ЛПР в ТК
  2. Получить разрешение ЛПР на проведение работы в ТК
    • Разрешение на проведение полной инвентаризации
    • Разрешение на осуществление выкладки продукции
Разрешение ЛПР на проведение полной инвентаризации.

Регулярный

  1. Поздороваться с ЛПР.
  2. При соблюдении договоренностей: "Благодарим" за соблюдение договоренностей.
  3. При не соблюдении договоренностей: Выяснение причины по наблюдению ранее установленной договоренности и произвести повторное согласование.
Разрешение ЛПР на проведение полной инвентаризации.

4. Снятие остатков

Первичный

Методы снятия остатков:

  1. Запросить остатки у в ЛПР с компьютера с помощью предоставления PLU по продукции. (Наилучший способ для повышения лояльности со стороны ЛПР)

Регулярный

  1. Запросить остатки у в ЛПР с компьютера с помощью предоставления PLU по продукции.
  2. Запросить ТСД для снятия остатков по ШК.
  3. Использовать сканер ШК продукции на столбе информации.

5. Полная/Частичная инвентаризация

Первичный

  • Осуществить пересчёт всех остатков по продукции во всех местах хранения
    1. ОМП (ДМП)
    2. КХ
  • Определить количество брака по каждому SKU.
  • Предоставить ЛПР данные по браку и продукции с истёкшими сроками годности для последующего списания.
    Списание продукции производится сразу после подачи информации ЛПР в присутствии МЧ.
  • Выяснить в ТК причину дефростации продукции.

Регулярный

  • Осуществить пересчёт позиций с низким товарным запасом
    При остатке товара менее 5 штук и отсутствии продаж более 2 дней, договориться с ЛПР о списании товара.
  • Зафиксировать информацию по количеству брака по каждому SKU.
  • Провести локальную инвентаризации по браку и товару с истёкшим сроком годности.
    Списание продукции производится сразу после подачи информации ЛПР в присутствии МЧ.

6. Переговоры с ЛПР

Первичный

  1. После определения лучшего места продаж в ТК, договориться с ЛПР о выкладке продукции в данных местах продаж.

Регулярный

  1. Контроль ранее достигнутых договорённостей с ЛПР по местам продаж
  2. Проведение переговоров с ЛПР о расширении представленности ТМ Айсберри/Петрохолод на местах продаж
  3. Проведение переговоров с ЛПР об установке дополнительных мест продаж (ДМП)

7. Выкладка продукции

Первичный

Осуществление выкладки согласно ранее согласованных договоренностей с ЛПР м соблюдением стандартов компании.
